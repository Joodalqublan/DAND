(fordgobike Dataset Exploration)¶

Introduction¶

In this presentation, I will show my analysis of Ford gobike for the period (January 2018).

The main features I was interested in the dataset is :¶

  • I want to see which time of month the user start.
  • Explore if there a relationship between Age - Gender - User type .
  • Then, analyze if it's effected on duration time.

First Exploration Insight :¶

ford gobike users Start was increased over the month.¶
